# ToDo
A To-Do list application for iOS.

## Tech/Frameworks Used
- Swift
- UIKit (100% Programmatic UI)
- CoreData

## Features
- Supports dark mode and light mode
- Add new tasks
- Edit existing tasks
- Remove tasks
- Mark tasks as completed
- View date and time task was created
- Haptic feedback
